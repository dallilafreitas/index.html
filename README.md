# Frontend Website – HTML, CSS & JavaScript

Projeto de site frontend responsivo desenvolvido com HTML, CSS e JavaScript puro, com deploy automático via GitHub Actions.

##  Tecnologias
- HTML5
- CSS3
- JavaScript (Vanilla)
- GitHub Actions
- GitHub Pages

## CI/CD
Este projeto utiliza GitHub Actions para:
- Validar a estrutura do projeto
- Realizar deploy automático a cada push na branch `main`
